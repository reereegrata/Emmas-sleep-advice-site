## Optimization Rules (To Prevent Over-Optimization)

1. **One pass per page per month** — unless a critical bug is found (broken link, missing schema, wrong canonical)
2. **Meta title:** Only trim if over 71 chars. Do NOT change content/topic.
3. **OG tags:** Only fix if duplicate or missing. Do NOT change content.
4. **Internal links:** Already 3-6 per guide. Do NOT add more.
5. **External links:** All guides have Red Nose links. Do NOT add more.
6. **Schema:** Present on all pages. Do NOT add more types unless needed for new content.
7. **Content text:** Do NOT re-optimize already written content unless GSC data shows a significant drop (>5 positions).
8. **Before suggesting ANY optimization:** Check this file. If the page/optimization is listed, SKIP.
